## Description
A Python console-based cafe management system that allows users to browse menus,
add and remove items from a cart, and calculate the total bill.

## Features
- Menu categories: Beverages, Desserts, Snacks
- Add and remove items from cart
- Total bill calculation
- Input validation using try/except
- Object-Oriented Programming

## Technologies Used
- Python 3
